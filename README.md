# 台湾関税統計 ダッシュボード生成ツール

財政部関税統計の原始データから、日本語版分析ダッシュボードを自動生成する Web ツール。

## 機能

- 関税統計の `.xls` をアップロード → 整形済み `.xlsx` を出力
- 品目情報・期間・国家リストを自動検出
- 輸入/輸出いずれにも対応
- HS code ごとの日文品名・単価市況レンジを管理
- 国名対照表 (英→日) を Web 上で編集可能
- 集計値はすべて SUMIFS 等の数式で総表から参照 (検証可能)

## ディレクトリ構造

```
dashboard_tool/
├── app.py                       # Streamlit メインページ
├── dashboard_generator.py       # 生成ロジック (純関数)
├── requirements.txt
├── data/
│   ├── country_mapping.json     # 国名対照
│   └── commodity_settings.json  # 品目設定
└── pages/
    ├── 1_🌏_国名対照.py          # 対照表編集ページ
    └── 2_🧪_品目設定.py          # 品目設定ページ
```

## ローカル起動

```bash
# 1. 依存をインストール
pip install -r requirements.txt

# 2. アプリ起動
streamlit run app.py
```

ブラウザで `http://localhost:8501` を開くと UI が表示されます。

## 社内ネットワーク公開

社内の常時稼働 PC で起動し、社員がアクセスできるようにする手順:

```bash
# PC の IP アドレスを指定して起動
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

社員は `http://[PCのIP]:8501` でアクセス可能。

### Windows でサービス化する場合

タスクスケジューラで PC 起動時に自動実行を設定:

1. タスクスケジューラを開く
2. 「タスクの作成」→ 名前を「ダッシュボードツール」
3. トリガー: 「ログオン時」
4. 操作: プログラム `streamlit`、引数 `run app.py --server.address 0.0.0.0`
5. 開始 (フォルダ): `C:\path\to\dashboard_tool`

## 使い方

### 1. ダッシュボード生成

1. メインページで `.xls` ファイルをアップロード
2. 検出された品目情報・期間を確認
3. 輸出入区分を選択 (両方ある場合)
4. 「ダッシュボード生成」をクリック
5. プレビュー確認後、ダウンロード

### 2. 国名対照の編集

「🌏 国名対照」ページでテーブルを直接編集。
新規国家が原始データに出現すると、ダッシュボード生成時に自動で末尾に追加されます。
あとでこのページで日文名を設定してください。

### 3. 品目設定の追加

「🧪 品目設定」ページで HS code・日文品名・単価妥当レンジを登録。
登録した品目は次回以降のダッシュボード生成で自動適用されます。

## データ更新時の挙動

ダッシュボードの集計値はすべて「総表」シートから SUMIFS で参照しているため、
**総表のデータを差し替えれば、ダッシュボード全体が自動再計算**されます。
新しい月の確定値が発表されたら、総表の該当行を更新するだけで OK。

## トラブルシューティング

### .xls ファイルが読み込めない

`xlrd>=2.0` をインストールしてください (`pip install xlrd`)。

### 日本語が文字化けする

システムに「Yu Gothic」フォントがインストールされていない場合に発生。
Excel で開けば問題ない場合がほとんど。
他のフォントを使う場合は `dashboard_generator.py` の FONT_* 定数を変更。

### ポート 8501 が使用中

`streamlit run app.py --server.port 8502` のように別ポートを指定。
